# Sales Analytics Dashboard

Technologies: Excel, Pivot Tables, Dashboards, CSV

## What’s included
- `data/sales_data.csv` — sample retail sales data (monthly records).
- `INSTRUCTIONS.md` — step-by-step to open the file, create pivot tables, and build the dashboard.
- `export_example.csv` — same as data for quick preview.

## How to use (quick)
1. Download the repository and open `data/sales_data.csv` in Excel.
2. Insert -> PivotTable -> Use the data range -> place on new sheet.
3. Create KPIs: Sum of Sales, Count of Orders, Avg Order Value.
4. Add Slicers for Region and Month, insert charts (line for monthly sales, bar for top products).
5. Save your finished workbook as `Sales_Dashboard_YourName.xlsx` and upload to GitHub.
